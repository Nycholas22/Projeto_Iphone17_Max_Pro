iPhone 17 Pro Max - Landing Page Premium

Este é um projeto de uma landing page de alta performance e visual sofisticado para a venda fictícia do iPhone 17 Pro Max. O site foi desenvolvido com foco em design moderno, animações suaves e uma experiência de usuário (UX) de nível premium.

🛠️ Tecnologias Utilizadas

O projeto utiliza tecnologias de ponta para garantir rapidez e responsividade:

HTML5: Estruturação semântica do conteúdo.

Tailwind CSS: Framework de CSS utilitário para estilização rápida, responsiva e customização de cores (como o Laranja Cósmico).

Font Awesome: Biblioteca de ícones para elementos visuais intuitivos.

Google Fonts (Inter): Tipografia moderna e legível utilizada pela indústria de tecnologia.

CSS Animations: Keyframes customizados para efeitos de "Fade-Up" e transições de hover.

📱 Funcionalidades do Site

Navegação Inteligente: Barra de navegação fixa com efeito de desfoque (backdrop-blur) que facilita o acesso às seções em qualquer ponto da página.

Seção Hero de Alto Impacto: Título com gradiente dinâmico e botões de chamada para ação (CTA) com sombras suaves.

Grid de Especificações: Cards interativos que detalham o Hardware (Chip A19 Pro, Câmera, Tela e Bateria), com efeitos de elevação ao passar o mouse.

Seção de Inteligência: Destaque para o sistema de resfriamento e IA, utilizando layouts de grid assimétricos.

Seletor de Cores: Galeria interativa no rodapé permitindo visualizar as variações: Laranja Cósmico, Titânio, Noite e Abismo.

Responsividade Total: Design adaptado para visualização perfeita em computadores, tablets e smartphones.

🎨 Design e Estética

O site segue a linguagem visual "Dark Mode" da Apple, utilizando:

Contraste Alto: Fundo preto puro (#000000) com textos em branco e cinza suave.

Destaques Vibrantes: Uso estratégico da cor laranja para guiar o olhar do usuário para elementos importantes.

Espaçamento Amplo: Amplo uso de white space (espaço negativo) para transmitir uma sensação de luxo e organização.

📂 Como Executar Localmente

Faça o download do arquivo index.html.

Certifique-se de estar conectado à internet (necessário para carregar o Tailwind e os ícones via CDN).

Abra o arquivo em qualquer navegador moderno (Chrome, Edge, Firefox ou Safari).
